CSS サンプル

変数を扱う：var

                        
                            :root {
                                --font-color: #333;
                            }

                            body {
                                color: var(--font-color);
                            }

最小値、最大を設定できる：min(),max(),clamp()

                        
                            .clamp-box {
                                width: min(95%,840px);
                                font-size: clamp(1rem,3vw,3rem);
                            }
ここが「.clamp-box」

imgを背景画像のように扱う：object-fit

                        
                            .object-fit-cover {
                                width: 100%;
                                height: 100%;
                                object-fit: cover;
                            }
                            .object-fit-contain {
                                width: 100%;
                                height: 100%;
                                object-fit: contain;
                            }

背景・親要素と混合させる：mix-blend-mode

                        
                            .mix-blend-mode-text {
                                mix-blend-mode: overlay;
                            }

Moon

自由に要素を切り抜く：clip-path

                        
                            .clip-path {
                                max-width: 400px;
                                height: 400px;
                                margin-top: 1rem;
                                background: url(../img/zoom3.jpeg) no-repeat center/cover;
                                clip-path: polygon(51% 9%, 70% 34%, 96% 36%, 71% 61%, 84% 85%, 49% 69%, 21% 88%, 29% 62%, 10% 34%, 38% 34%);
                            }

コンテンツ幅になるwidth：content-fit

                        
                            .fit-content-text {
                                    width: fit-content;
                                    margin: auto;
                                    border: 1px solid #999;
                                    font-size: 3rem;
                            }
